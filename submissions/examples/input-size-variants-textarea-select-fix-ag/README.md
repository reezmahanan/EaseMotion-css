# Input Size Variants Extension Fix (Issue #14075)\n\n## What does this do?\nDemonstrates size modifier classes (`.ease-input-sm`, `.ease-input-lg`) extended to target `.ease-textarea` and `.ease-select` in addition to `.ease-input`.\n\n## How is it used?\n```html\n<textarea class="ease-textarea ease-input-sm"></textarea>\n<select class="ease-select ease-input-lg">...</select>\n```\n\n## Why is it useful?\nDevelopers expect size modifiers to work consistently across all form control types. Without this fix, textarea and select elements ignore the size classes entirely.
